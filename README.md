这是一个教师节专题新闻稿网页，记录了百色中学2406第五小组对刘中江物理老师的采访。

## 快速访问

- 采访主题：纸墨传情，访恩师心声
- 采访对象：刘中江物理老师（从教28年）
- 采访团队：百色中学2406第五小组
- 制作团队：
